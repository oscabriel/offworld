New to development? Start here:

1. **[Project Setup](/developer-guide/setup)** - Install dependencies, configure environment
2. **[Development Workflow](/developer-guide/development-workflow)** - Run dev server, make changes
3. **[Contributing](/developer-guide/contributing)** - Submit your first PR

## Architecture Overview

Offworld uses a **triple-deploy architecture**:

- **Backend**: Convex Cloud (database, workflows, RAG)
- **Frontend**: Cloudflare Workers (edge SSR)
- **Docs**: Netlify (static site)

Learn more in the [Architecture Deep Dive](/developer-guide/architecture).
