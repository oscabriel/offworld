## Repositories API

### `repos.getByFullName`

**Type**: Query

**Args**:
```typescript
{ fullName: string } // "facebook/react"
```

**Returns**:
```typescript
{
  _id: Id<"repositories">,
  fullName: string,
  owner: string,
  name: string,
  stars: number,
  language: string,
  description: string,
  summary?: string,
  architecture?: string,
  mermaidDiagram?: string,
  status: "queued" | "processing" | "completed" | "failed",
  lastAnalyzedAt?: number
} | null
```

**Usage**:
```tsx
const repo = useQuery(api.repos.getByFullName, {
  fullName: "facebook/react"
});
```

### `repos.startAnalysis`

**Type**: Mutation

**Auth**: Required

**Args**:
```typescript
{ fullName: string }
```

**Returns**: `Id<"repositories">`

**Usage**:
```tsx
const start = useMutation(api.repos.startAnalysis);
await start.mutateAsync({ fullName: "facebook/react" });
```

### `repos.listIndexed`

**Type**: Query

**Args**: None

**Returns**:
```typescript
Array<{
  fullName: string,
  stars: number,
  language: string,
  lastAnalyzedAt: number
}>
```

**Usage**:
```tsx
const indexed = useQuery(api.repos.listIndexed);
```

## Chat API

### `chat.getConversation`

**Type**: Query

**Args**:
```typescript
{ conversationId: Id<"conversations"> }
```

**Returns**:
```typescript
{
  _id: Id<"conversations">,
  repoId: Id<"repositories">,
  title: string,
  messages: Array<{
    role: "user" | "assistant",
    content: string,
    toolCalls?: Array<{ name: string, args: any }>
  }>
}
```

### `chat.sendMessage`

**Type**: Mutation

**Auth**: Required

**Args**:
```typescript
{
  conversationId: Id<"conversations">,
  message: string
}
```

**Returns**: `Id<"messages">`

**Usage**:
```tsx
const send = useMutation(api.chat.sendMessage);
await send.mutateAsync({
  conversationId,
  message: "How does auth work?"
});
```

### `chat.createConversation`

**Args**:

## Error Codes

### `UNAUTHORIZED`

User not authenticated. Call `authClient.signIn.social({ provider: "github" })`.

### `NOT_FOUND`

Resource doesn't exist (repo, issue, PR, etc.).

### `RATE_LIMIT`

GitHub API rate limit exceeded. Wait or use GitHub App.

### `VALIDATION_ERROR`

Invalid input args. Check arg types.

### `ANALYSIS_FAILED`

Workflow failed. Check Convex logs for details.
